**NOTA:** OCR com erros — "Firmimo Vaz Poda" = Firmino Vaz Sodré; "Laurinda Paz" = Laurinda Vaz. Este é o **Firmino Vaz Sodré** já identificado nos inventários APEB (Amargosa). A transcrição "Poda/Paz" é claramente erro de OCR para "Sodré/Vaz".

**RELEVÂNCIA GENEALÓGICA:**
- Firmino Vaz Sodré = membro do ramo Vaz Sodré de Amargosa
- Laurinda Vaz = possivelmente Laurinda Vaz Sodré (mesma família)
- O registro é de **1909**, portanto Firmino já era adulto/casado neste período
- Corrobora a presença do ramo Vaz Sodré em Amargosa no início do séc. XX
- Firmino Vaz Sodré aparece nos inventários APEB como inventariante
